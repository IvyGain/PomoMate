## 📋 次の具体的アクション

### 今週やること
1. **環境セットアップ**
   ```bash
   # バックエンド起動
   cd backend
   docker-compose up -d
   npm run dev
   
   # フロントエンド更新
   cd ../
   npm install axios socket.io-client
   ```

2. **APIクライアント作成**
   - `src/services/api.js` - API基底クラス
   - `src/services/auth.js` - 認証API
   - `src/services/session.js` - セッションAPI

3. **認証フロー実装**
   - authStoreをAPIと連携
   - トークン管理実装
   - 自動ログイン実装
