# 📱 PomoMate - スマホテストガイド

## 🚀 スマホでのテスト手順

### ステップ 1: Expo Go アプリのインストール
1. **iPhone**: App Store で「Expo Go」を検索してインストール
2. **Android**: Google Play で「Expo Go」を検索してインストール

### ステップ 2: 開発サーバーの起動

ターミナルで以下のコマンドを実行：
```bash
cd "/Users/mashimaro/Documents/Workspace/PomoMate"
npx expo start --tunnel --clear
```

または：
```bash
./start-demo.sh
```

### ステップ 3: アプリにアクセス

#### 方法 1: QRコードスキャン（推奨）
1. Expo Goアプリを開く
2. 「Scan QR Code」をタップ
3. ターミナルに表示されるQRコードをスキャン

#### 方法 2: 直接URL入力
1. Expo Goアプリを開く  
2. ターミナルに表示される `exp://` で始まるURLを入力

#### 方法 3: Webブラウザ（PCのみ）
- http://localhost:8081 にアクセス
