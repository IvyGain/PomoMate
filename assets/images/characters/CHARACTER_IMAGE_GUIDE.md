# キャラクター画像作成ガイド

## 概要
本ドキュメントは、PomoMateアプリで使用するキャラクター画像の作成ガイドラインです。

## 画像仕様

### 基本仕様
- **フォーマット**: PNG (透過背景推奨)
- **サイズ**: 512x512px (正方形)
- **解像度**: 72dpi以上
- **カラーモード**: RGB
- **ファイルサイズ**: 1MB以下推奨

### 命名規則
```
{character_id}.png

例:
- balanced_1.png
- focused_2.png  
- consistent_3.png
- ninja_focused_3.png
```

## デザインガイドライン

### 全体的な方針
1. **シンプルで親しみやすいデザイン**
   - 丸みを帯びたフォルム
   - 大きな目と表情豊かな顔
   - 子供から大人まで親しめるデザイン

2. **レベルによる成長表現**
   - レベル1: 小さく可愛らしい（たまご型、小動物風）
   - レベル2: 少し成長（子供風）
   - レベル3: さらに成長（ティーン風）
   - レベル4: 成熟（大人風、威厳がある）
   - レベル5: 伝説級（神秘的、オーラがある）

3. **タイプ別の特徴**
   - **バランス型**: 緑系、丸みのある形、優しい表情
   - **集中型**: 青系、シャープな形、真剣な表情
   - **継続型**: オレンジ系、安定感のある形、穏やかな表情

### キャラクター別デザイン要素

#### レベル1（初期キャラクター）
- 基本的に「たまご」型のフォルム
- 大きな目と小さな手足
- パステルカラー使用

#### レベル2（第一進化）
- 「こども」型のフォルム
- より人間的または動物的な特徴
- 少し濃いめの色使い

#### レベル3（第二進化）
- より複雑なデザイン
- 装飾品や特徴的なアイテムを持つ
- タイプの特徴が明確に

#### レベル4（第三進化）
- 威厳のあるデザイン
- エフェクトやオーラ表現
- 豪華な装飾

#### レベル5（最終進化）
- 神話的・伝説的な要素
- 光や特殊効果
- 圧倒的な存在感

### 特殊キャラクターのデザイン
- **忍者系**: 黒・紺色、シャープなライン、忍者装束
- **侍系**: 赤・紺色、鎧や刀、威厳のある姿勢
- **龍系**: 赤・金色、鱗、炎のエフェクト
- **鳳凰系**: オレンジ・赤、羽、再生の象徴
- **狐系**: 紫・白、九尾、神秘的な雰囲気

## 制作時の注意点

1. **統一感の維持**
   - 全キャラクターで線の太さを統一
   - 影の付け方を統一
   - 目の描き方を統一

2. **視認性の確保**
   - 小さいサイズでも識別可能
   - 背景との十分なコントラスト
   - シルエットだけでも識別可能

3. **アニメーション考慮**
   - 将来的なアニメーション実装を考慮
   - パーツの分離を意識
   - シンプルな構造

## 納品チェックリスト

- [ ] PNG形式、透過背景
- [ ] 512x512pxの正方形
- [ ] ファイル名は命名規則に準拠
- [ ] 1MB以下のファイルサイズ
- [ ] キャラクターIDと一致
- [ ] タイプ別の色使いが適切
- [ ] レベルに応じた成長表現
