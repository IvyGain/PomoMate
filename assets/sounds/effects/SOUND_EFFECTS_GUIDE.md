# 効果音作成ガイド・要件定義書

## 概要
本ドキュメントは、PomoMateアプリで使用する効果音の要件定義と作成ガイドラインです。

## 音声ファイル仕様

### 基本仕様
- **フォーマット**: MP3またはWAV
- **サンプリングレート**: 44.1kHz
- **ビットレート**: 128kbps以上（MP3の場合）
- **ビット深度**: 16bit以上（WAVの場合）
- **チャンネル**: モノラルまたはステレオ
- **音量**: -12dB〜-6dB（ピーク値）
- **長さ**: 0.1秒〜3秒（用途により異なる）

### 命名規則
```
{action}_{type}.mp3

例:
- timer_start.mp3
- timer_complete.mp3
- level_up.mp3
- game_correct.mp3
```

## 音響デザインの方針

### 全体的なトーン
- **ポジティブで親しみやすい**: ユーザーのモチベーションを維持
- **疲れない音質**: 長時間使用しても不快にならない
- **統一感**: 全体的に調和の取れた音のセット

### 音量バランス
- 通知系: 基準音量（-9dB）
- 達成系: やや大きめ（-6dB）
- 操作系: 小さめ（-12dB）
- エラー系: 基準音量（-9dB）

### 実装上の考慮事項
1. **遅延**: 100ms以下の再生遅延を目指す
2. **重複再生**: 同じ音の連続再生に対応
3. **音量調整**: アプリ内で音量調整可能にする
4. **ミュート**: 個別の効果音ON/OFF設定
